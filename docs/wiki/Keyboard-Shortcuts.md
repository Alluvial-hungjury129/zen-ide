# Keyboard Shortcuts — Complete Reference

All shortcuts use **Cmd** (⌘) on macOS and **Ctrl** on Linux.

## File Operations

| Shortcut | Action |
|---|---|
| `Cmd+N` | New file |
| `Cmd+O` | Open file |
| `Cmd+Shift+O` | Open workspace folder |
| `Cmd+S` | Save file |
| `Cmd+Shift+S` | Save as |
| `Cmd+W` | Close current tab |
| `Cmd+Q` | Quit Zen IDE |

## Editing

| Shortcut | Action |
|---|---|
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+X` | Cut line or selection |
| `Cmd+C` | Copy line or selection |
| `Cmd+V` | Paste |
| `Cmd+A` | Select all |
| `Cmd+/` | Toggle line comment |
| `Cmd+]` | Indent selection |
| `Cmd+[` | Unindent selection |

## Find & Replace

| Shortcut | Action |
|---|---|
| `Cmd+F` | Find & replace (in file) |
| `Cmd+G` | Find next match |
| `Cmd+Shift+G` | Find previous match |
| `Cmd+Shift+F` | Search across all files (global search) |

## Navigation

| Shortcut | Action |
|---|---|
| `Cmd+P` | Quick open — fuzzy file finder |
| `Cmd+Shift+E` | Focus file explorer |
| `Cmd+Click` | Go to definition |
| `Cmd+Hover` | Underline navigable symbols |
| `Cmd+Space` | Trigger autocomplete |

## Cursor Movement

| Shortcut | Action |
|---|---|
| `Opt+Left` / `Cmd+Left` | Move to line start (smart home) |
| `Opt+Right` / `Cmd+Right` | Move to line end |
| `Cmd+Backspace` | Delete to line start |

## AI Features

| Shortcut | Action |
|---|---|
| `Tab` | Accept inline AI suggestion |
| `Escape` | Dismiss AI suggestion |
| `Alt+]` | Next AI suggestion |
| `Alt+[` | Previous AI suggestion |
| `Cmd+Right` | Accept suggestion word-by-word |
| `Cmd+H` | AI chat history |

## View & Panels

| Shortcut | Action |
|---|---|
| `` Cmd+` `` | Focus terminal |
| `Cmd+K` | Clear terminal |
| `Cmd+.` | Toggle Dev Pad |
| `Cmd+Shift+D` | Toggle Sketch Pad |
| `Cmd+Shift+T` | Theme picker |
| `Cmd+Shift+L` | Toggle dark/light mode |
| `Cmd+R` | Hot reload IDE |
| `Cmd+Shift+I` | Widget inspector (DevTools mode) |
| `Cmd+Shift+/` | Show all keyboard shortcuts |
| `F10` | Open app menu |

## Layout

| Shortcut | Action |
|---|---|
| `Cmd+Shift+\` | Maximise focused panel |
| `Cmd+Shift+0` | Reset layout / Maximise window |
| `Cmd+Shift+U` | Toggle all UI panels |

## Zoom

| Shortcut | Action |
|---|---|
| `Cmd++` | Zoom in |
| `Cmd+-` | Zoom out |
| `Cmd+0` | Reset zoom |

## Terminal

| Shortcut | Action |
|---|---|
| `Cmd+C` / `Ctrl+Shift+C` | Copy selection |
| `Cmd+V` / `Ctrl+Shift+V` | Paste |
| `Opt+Left` | Move cursor back one word |
| `Opt+Right` | Move cursor forward one word |
| `Opt+Backspace` | Delete back one word |
| `Cmd+Click` | Open file path from terminal output |
| `Ctrl+C` | Interrupt running command |

## Popup / Dialog Navigation

Most popups support vim-style navigation:

| Key | Action |
|---|---|
| `j` / `↓` | Move down |
| `k` / `↑` | Move up |
| `Enter` | Select / Confirm |
| `Escape` / `q` | Close |
| `Tab` | Switch focus |
| `Ctrl+N` / `Ctrl+P` | Next / Previous item |
